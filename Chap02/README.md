## 02-3 가변 변수와 불변 변수

* 책의 진행은...불변 변수로 정의된 배열이나 객체를 내장 함수로 수정하는 것을 암묵적으로 금지하여 무결성을 유지한다고 함.
  * 가급적 불변 변수만 사용해서 실습을 진행할 예정...

| 가변 내장 함수         | 무결성 내장 함수                           |
| ---------------------- | ------------------------------------------ |
| push(...items)         | concat(...items)                           |
| splice(s, c, ...items) | slice(0, s).concat(...items, slice(s + c)) |
| pop()                  | slice(0, len -1)                           |
| shift()                | slice(1)                                   |
